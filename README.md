# ROS-Lab

Turtlebot manual: <https://emanual.robotis.com/docs/en/platform/turtlebot3>

Install `docker`

* <https://docs.docker.com/engine/install/debian>

Install `rocker`

* `uv tool install rocker`

Get `gazebo_models`

* `make get_gazebo_models`

Build custom ros docker image

* `make ros_image_build`

Spin up ros container

* `make ros_up`

``Prefix`` + ``I``

where Prefix = ``CTRL`` + ``S``
